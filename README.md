# avro-playground

**run gradle build.**

~~~sh
$ ./gradlew build
~~~

**run gradle tests.**

~~~sh
$ ./gradlew test
~~~

**dependency versions:**

- [kotlin 1.5.21](https://github.com/JetBrains/kotlin/releases/tag/v1.5.21)
- [gradle 7.2](https://github.com/gradle/gradle/releases/tag/v7.2.0)
- [junit 5.7.2](https://github.com/junit-team/junit5/releases/tag/r5.7.2)
